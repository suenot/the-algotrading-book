# The Algotrading Book: Machine Learning for Crypto Markets

> 🇷🇺 [Читать на русском](README.ru.md)

A comprehensive 365-chapter course — one chapter per day for a year — covering the full spectrum of machine learning methods applied to cryptocurrency trading. All implementations use the **Bybit API** (REST + WebSocket) with **Python** and **Rust** examples.

---

## Structure

**365 chapters · 23 thematic blocks · 12 months**

Each chapter contains:
- Theoretical foundations with mathematical derivations
- Python implementation (Bybit API v5)
- Rust implementation (tokio + reqwest)
- 3 practical trading examples
- Backtesting framework
- Performance evaluation metrics
- Future directions & references
